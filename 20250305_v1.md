## 🤖 Agent架构与应用（优先展示）

### 1. Alibaba OpenSandbox - 通用AI沙盒平台开源
**Summary**: 阿里巴巴开源OpenSandbox，一个面向AI应用的通用沙盒平台，支持多语言SDK、统一沙盒API和Docker/Kubernetes运行时。主要应用于Coding Agents、GUI Agents、Agent评估、AI代码执行和RL训练等场景。提供Python、Java/Kotlin、JavaScript/TypeScript、C#/.NET等多种SDK，内置命令、文件系统和代码解释器环境，支持浏览器自动化和桌面环境。
**原文地址**: https://github.com/alibaba/OpenSandbox

### 2. Cognition发布Devin 2.2重大更新
**Summary**: Cognition发布Devin 2.2，这是自发布以来最重要的更新。Devin作为云Agent平台，可以像团队成员一样工作——分配任务、审查PR、处理积压工作。Cognition内部使用Devin来构建Devin本身。此前还发布了SWE-1.5模型，与Cerebras合作实现高达950 tok/s的推理速度，以及Agent Trace开放规范用于记录AI贡献。
**原文地址**: https://www.cognition.ai/blog

### 3. Anthropic工程博客：多并行Claude构建C编译器
**Summary**: Anthropic工程团队分享如何使用并行Claude团队构建C编译器的技术文章。同时发布其他重要技术内容：AI Agent评估方法、长运行Agent的有效Harness设计、上下文工程技术、高级工具使用等。展示了如何构建有效的AI Agent系统和评估方法。
**原文地址**: https://www.anthropic.com/engineering/building-c-compiler

### 4. Amp编码Agent产品更新
**Summary**: Amp（前身为Claude Code团队）发布多项重要更新：(1) GPT-5.3-Codex模型支持Deep模式；(2) 推出"Deep"代理模式，思考更深入、规划更全面；(3) 新增可共享的代码走查Walkthroughs功能；(4) Agent Skills支持，可复用现有Agent技能；(5) Python SDK发布；(6) 移除Tab功能，认为这不是未来的方向。
**原文地址**: https://ampcode.com/chronicle

### 5. Claude Scientific Skills - 科研Agent技能库
**Summary**: K-Dense AI发布170+科研Agent技能库，支持Cursor、Claude Code、Codex等工具。涵盖生物信息学、化学信息学、蛋白质组学、临床研究、医疗AI、机器学习、材料科学、物理天文、工程仿真等多个领域。提供250+科学和金融数据库访问、60+优化Python包技能。
**原文地址**: https://github.com/K-Dense-AI/claude-scientific-skills

### 6. Manus (Meta) 近期更新汇总
**Summary**: Manus（已被Meta收购）近期推出多项功能：(1) Manus Agents集成Telegram；(2) 项目技能（Project Skills）功能，将团队专业知识转化为可复用资产；(3) 支持Agent Skills开放标准；(4) 应用共享和测试功能；(5) Manus Sandbox云计算机；(6) 与Similarweb合作提供数字营销专业知识；(7) Slack连接器；(8) Manus Design View设计视图。
**原文地址**: https://manus.im/zh-cn/blog
