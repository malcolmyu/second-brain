## 🤖 模型发布

### Anthropic 发布 Claude Sonnet 4.6
- **Summary**: Anthropic 发布 Claude Sonnet 4.6，这是自 Opus 4.6 发布仅 12 天后的重大升级。新模型支持 100 万 token 上下文窗口（是之前的 4 倍），在编码、指令遵循、计算机使用等方面有显著提升。在 SWE-Bench 和 OS World 基准测试中创下新纪录，ARC-AGI-2 得分 60.4%。
- **原文地址**: https://techcrunch.com/2026/02/17/anthropic-releases-sonnet-4-6/

### Google 发布 Gemini 3.1 Pro
- **Summary**: Google 推出 Gemini 3.1 Pro，这是 Gemini 3 系列的最新核心推理模型。在 ARC-AGI-2 基准测试中得分 77.1%，超越 Claude Opus 4.6（68.8%）和 GPT-5.2（52.9%）。该模型支持原生多模态输入，在逻辑推理、知识理解和创造性编码方面表现卓越。
- **原文地址**: https://www.cnet.com/tech/services-and-software/google-announces-gemini-3-1-pro/

### Cognition 发布 SWE-1.6 预览版
- **Summary**: Cognition 分享 SWE-1.6 训练运行的早期预览。作为专为软件工程优化的模型系列最新成员，SWE-1.6 在 SWE-Bench Pro 上性能提升 11%，训练速度比前代提升 6 倍。模型与 Cerebras 合作，提供高达 950 tok/s 的推理速度。
- **原文地址**: https://cognition.ai/blog/swe-1-6-preview

## 📚 学术研究

### SkillsBench: 评估 Agent 技能效果的基准测试
- **Summary**: 该研究提出了一个基准测试，用于衡量精心策划的 Skills 与自生成 Skills 在多样化任务中对 Agent 成功率的影响。研究评估了 7 种模型-工具配置在 7308 个轨迹上的表现，识别出影响成功或失败的关键因素。
- **原文地址**: https://arxiv.org/abs/2602.12670

### Think Deep, Not Just Long: 通过深度思考 Token 衡量 LLM 推理努力
- **Summary**: 论文定义了深度思考比率（DTR），通过计算仅在深层网络中预测分布才收敛的 token 数量来衡量模型推理深度。研究表明这一指标与准确率的相关性优于 token 长度或置信度，可用于指导更高效的推理。
- **原文地址**: https://arxiv.org/abs/2602.13517

### BitDance: 使用二进制 Token 扩展自回归生成模型
- **Summary**: 该模型使用紧凑的二进制视觉 token 表示图像，并结合基于扩散的技术，以比同类自回归方法更高效、更快速的方式生成高分辨率图像。
- **原文地址**: https://arxiv.org/abs/2602.14041

### WebWorld: 用于 Web Agent 训练的大规模世界模型
- **Summary**: 该模型基于超过 100 万个真实网页交互轨迹训练，使用可扩展的分层流程实现长程模拟、多格式输入和改进的 Agent 在基准测试及下游 Web 任务上的表现。
- **原文地址**: https://arxiv.org/abs/2602.14721
