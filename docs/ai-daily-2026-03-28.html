LangChain Blog · 2026-03-26

How we build evals for Deep Agents

我们如何为 Deep Agents 构建评估 文章阐述了评估在引导 Deep Agents 行为中的关键作用，并提出了构建高质量评估的实践方法。 明确行为目标：先确定生产环境中真正重要的行为，如跨文件检索或连续调用多个工具。 针对性评估：针对这些行为精心挑选并设计评估，避免盲目堆砌大量测试。 自说明文档：为每个评估添加 docstring，说明其衡量的能力，并使用标签（如 tool_use）便于分组运行。 持续追踪与改进：所有评估运行记录在 LangSmith 项目中，团队可随时分析失败模式、更新覆盖范围并重新评估评估价值。 通过上述流程，评估既能真实反映生产需求，又能在迭代中不断提升代理的准确性和可靠性。
