Skills in LangSmith Fleet

LangSmith Fleet 现在支持共享技能，使团队成员能够为代理配备特定任务的专业知识。这些技能可以通过提示创建，也可以手动创建，还可以从模板或之前的代理对话中生成。共享的技能会自动同步到工作空间，确保所有代理都能访问最新的公司知识。 文章指出，代理的有效性很大程度上依赖于领域知识，而这些知识往往分散在团队成员的头脑和各种文档中。技能的出现旨在将这些隐性知识显性化，使其成为代理可用的资源。一个技能可以包含一系列指令和领域知识，例如支持工单分类标准、品牌语调指南或退款处理流程。代理仅在需要时加载相关技能，从而保持专注和响应迅速。这不仅能帮助新成员更快上手，还能在员工离职时保留宝贵的知识资产。Fleet 提供了多种创建技能的方式，包括利用 AI 进行创建。

How Moda Builds Production-Grade AI Design Agents with Deep Agents

Moda 是一款面向非设计专业人士的 AI 原生设计平台，旨在帮助营销人员、创始人等用户无需设计背景即可创建专业级演示文稿、社交媒体帖子等。 该平台的核心是一个基于 **Deep Agents** 构建的多智能体系统，并利用 **LangSmith** 提供可观测性，以支持快速迭代和可靠发布。 Moda 面临的挑战在于，与代码生成不同，视觉设计缺乏成熟的抽象表示，导致 AI 难以生成高质量、美观的设计。 Moda 的 AI 系统包含三个智能体： 1. **设计智能体**：负责在画布上实时创建和迭代设计。 2. **研究智能体**：从外部源提取结构化内容。 3. **品牌套件智能体**：整合品牌资产，确保设计符合品牌调性。 其中，研究智能体和品牌套件智能体运行在 **Deep Agents** 上，而设计智能体目前运行在自定义的 LangGraph 循环上，团队正考虑将其迁移至 Deep Agents。

letta-ai/claude-subconscious

Claude Subconscious 是一个实验性的后台代理，旨在解决 Claude Code 在会话间遗忘的问题。它通过 Letta 的内存系统、工具访问和上下文工程来增强 Claude Code 的能力。 该代理会监控 Claude Code 的会话记录，并读取用户的代码库，利用 Read、Grep 和 Glob 等工具进行探索。它能够跨会话、项目和时间累积记忆，并在每次提示前“低语”提供上下文、模式和提醒，从而引导 Claude Code。Claude Subconscious 运行在后台，不会阻塞用户，并且可以通过 Letta 的 Conversations 功能实现跨多个 Claude Code 会话的共享内存。其工作流程是在每次 Claude Code 响应后，将对话记录发送给 Letta 代理，代理更新内存并准备下一次的提示。 文章同时推荐了完全开源、内存优先且模型无关的 Letta Code 作为替代方案。
