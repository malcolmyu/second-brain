核心决策模型

🎯
第一性原理
亚里士多德 / 马斯克 / 芒格
决策 产品 认知
定义
从最基本的真理出发，通过逻辑推理得出结论的思维方式。剥离表象，追问事物本质，从根本层面解决问题而非修修补补。
核心洞察
  • 用户不是自然人，而是需求的集合——这是产品经理的第一性原理
  • 避免功能堆砌，回归用户本质需求
  • 把自己产品化，是职业发展的第一性原理
关联模型
批判性思维 奥卡姆剃刀 系统思考 黄金圈法则
🤔
批判性思维
《学会提问》/ 芒格思维模型
认知 决策
定义
一种理性的、反思性的思维方式，旨在决定我们信什么和做什么。核心是不盲从权威，独立思考，审慎判断。包括：识别假设、评估证据、分析论证、反思结论。
核心洞察
  • 沟通时关注"为什么"多于"怎么做"——批判性思维的体现
  • 对系统架构有深度理解——不盲从表面现象，追问本质
🔄
逆向思维
查理·芒格 / 思维模型
决策 认知
定义
从反面思考问题的方法：不是思考如何成功，而是思考如何避免失败。芒格说："我只想知道我会死在哪里，这样我就永远不会去那里。"
核心洞察
  • 产品决策中，先思考什么会导致产品失败，然后避开这些坑
  • 批判性思维的重要工具：质疑常规思路
🕸️
系统思考
彼得·圣吉 / 芒格思维模型
系统思考 决策
定义
看待问题的整体性方法，关注事物之间的相互联系和反馈回路，而非孤立的事件。核心概念包括：存量与流量、增强回路、调节回路、延迟效应。
核心洞察
  • 对系统架构有深度理解——系统思考的实践
  • 产品即交易：理解产品生态系统的各要素互动
⚖️
二八定律（帕累托法则）
维尔弗雷多·帕累托 / 芒格
决策 效率
定义
80%的结果来自20%的原因。应用于时间管理、产品设计、投资决策。关键是识别那关键的20%。
核心洞察
  • 产品功能不是越多越好，找到20%的核心功能做到极致
  • 把自己产品化，就是找到那20%能带来80%价值的独特能力

更多决策模型

  • SWOT分析 — 战略分析框架：Strengths（优势）、Weaknesses（劣势）、Opportunities（机会）、Threats（威胁）
  • 10/10/10法则 — 长期决策思维：这个决定在10分钟后、10个月后、10年后会带来什么影响？
  • 5W1H分析法 — 六何分析法：What、Why、Who、When、Where、How
  • 六顶思考帽 — 平行思维工具：白帽（事实）、红帽（情感）、黑帽（风险）、黄帽（收益）、绿帽（创意）、蓝帽（控制）
  • 整合思维模型 — 面对复杂决策时，创造性地整合多方观点，找到超越二元对立的第三种方案
