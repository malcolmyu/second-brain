## 核心模块

### 1. Research - 深度调研报告

**定位**：小马的核心能力，对标林克的"30分钟深度报告"

**输入**：
```
"帮我调研 字节跳动 AI 布局"
"深度分析 {公司/技术/趋势}"
```

**输出**：
- Markdown 报告：`workspace/reports/YYYY-MM-DD-{topic}.md`
- 结构：核心洞察 → 背景定义 → 多维度分析 → 结论 → 参考来源

**质量标准**：
- 必须有核心洞察（3-5条），不是信息罗列
- 必须有数据或事实支撑
- 必须回答"为什么"和"意味着什么"
- 来源必须可追溯

**使用方式**：
```bash
openclaw skill research-report <调研主题>
```

---

### 2. Knowledge - 知识体系构建

**定位**：个人知识管理，解决"读了很多书但知识散落"的问题

**功能**：
1. **概念卡片**：每个知识点结构化存储
2. **关联网络**：自动发现概念间的连接
3. **知识 gaps**：追踪待补充的概念
4. **主题查询**：快速检索某一主题的所有思考

**存储结构**：
```
memory/knowledge-base/
├── index.json              # 概念索引
├── concepts/               # 概念卡片目录
│   ├── 第一性原理.json
│   ├── 批判性思维.json
│   └── ...
├── connections.md          # 知识图谱
└── gaps.md                 # 知识 gaps 追踪
```

**使用方式**：
```bash
# 添加知识
knowledge_cli.py add "标题" "内容" --tags "标签1,标签2"

# 查询知识
knowledge_cli.py show           # 显示所有概念
knowledge_cli.py search "关键词" # 搜索概念
knowledge_cli.py graph          # 生成知识图谱
knowledge_cli.py gaps           # 查看知识 gaps
```
