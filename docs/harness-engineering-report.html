一、核心概念与定义

Harness Engineering 是指构建系统围绕模型，使其成为有效的工作引擎。它不是简单地调用 LLM API，而是设计完整的 orchestration 系统来发挥模型的最大潜力。

1.1 Agentic 系统的分类

Anthropic 在其研究中明确区分了两类 Agentic 系统：

类型 定义 特点
Workflows LLM 和工具通过预定义的代码路径编排 可预测、一致性好、适合明确定义的任务
Agents LLM 动态指导自身进程和工具使用 灵活、自主决策、适合开放式问题

1.2 何时使用 Agentic 系统

"Success in the LLM space isn't about building the most sophisticated system. It's about building the right system for your needs."

Anthropic 建议遵循简单优先原则：

  1. 从最简单的解决方案开始
  2. 仅通过全面评估证明需要时才增加复杂性
  3. 先用检索和上下文示例优化单 LLM 调用
  4. 简单方案不足时才考虑多步骤 Agentic 系统
